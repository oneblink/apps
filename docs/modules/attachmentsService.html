uploadAttachment

  • uploadAttachment(options: UploadAttachmentConfiguration & { formId: number; onProgress?: ProgressListener }, abortSignal?: AbortSignal): Promise<FormSubmissionAttachment>
    • Defined in src/services/uploadAttachment.ts:51

    Upload a submission attachment. Attachment can be passed as a Blob, Uint8Array or string (base64). Will return data required form accessing the attachment.

    Example

     
    const blob = new Blob(['a string of data'], {
      type: 'text/plain',
    })
    const file = {
      formId: 1,
      data: blob,
      fileName: 'file.jpg',
      contentType: 'image/jpeg',
      isPrivate: true, // Whether the attachment will be able to be downloaded by other users
    }
    const abortController = new AbortController()
    const {
      s3: {
        key, // string
        bucket, // string
        region, // string
      },
      url, // string
      contentType, // string
      fileName, // string
      id, //string
      isPrivate, // boolean
      uploadedAt, // string
    } = await submissionService.uploadAttachment(
      file,
      abortController.signal,
    )

    Parameters

    • options: UploadAttachmentConfiguration & { formId: number; onProgress?: ProgressListener }
    • Optional abortSignal: AbortSignal

    Returns Promise<FormSubmissionAttachment>
